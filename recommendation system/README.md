# 🎬 Movie Recommendation System

A content-based Movie Recommendation System built using Python.

## Features

- Recommend Top 5 Similar Movies
- Content-Based Filtering
- Cosine Similarity Algorithm
- Simple Desktop GUI
- 100+ Movie Dataset

## Tech Stack

- Python
- Pandas
- Scikit-learn
- CustomTkinter

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
py app.py
```
